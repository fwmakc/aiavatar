# 6. 场景："库佳" — 家庭群聊中的家神

我们来配置一个温暖、关爱家庭群聊的机器人。库佳是一个家神，他认识所有家庭成员，给出建议，提醒注意健康，并发布有用的内容。

## 这个场景的工作方式

库佳被配置为**特定群聊的覆盖配置**。这意味着：
- `data/default.json` — 通用配置（来自第5章）
- `data/chats/{chatId}.json` — 库佳在家庭群聊中的配置

这样，同一个机器人可以在工作群聊中是马克斯，在家庭群聊中是库佳——拥有不同的人设。

## 第1步：获取家庭群聊的chatId

将机器人添加到家庭群聊中（具体方法见[第9章](09-running.md)）。

添加后，在群聊中发送任意消息。机器人控制台中会出现：
```
[Group] chat -1001234567890: ...
```

`chat` 后面的数字就是chatId（带负号）。文件名中**去掉负号**，只保留数字：`1001234567890`。

  "interests": "家庭温馨、烹饪与食谱、孩子与学习、园艺、家庭健康、民间俗语、家庭传统、节日、手工、宠物——猫和狗",

## 这里配置了什么

### 人设

- **名字：** 库佳 — 家神
- **风格：** 温暖、关爱，带着民间谚语和亲昵的称呼
- **personaStages：** 即使是敌对的库佳也会关心——他毕竟是家神，忍不住

### 内容

| 来源 | 提供内容 | 设置方式 |
|---|---|---|
| Bash.im | 轻松幽默 | 偶尔（权重 3）|
| JokeAPI (Misc) | 家庭笑话 | 较多（权重 4）|
| Film.ru | 电影新闻 | AI转述 |
| Afisha | 活动资讯 | AI转述 |

**测验** — 民间俗语、烹饪、谜语、园艺、节日。

**健康提醒** — 眼睛、背部、散步、健康睡眠、家庭活动。

### 日程安排

- **活跃时间：** 每天，07:00–22:00 — 从早上到深夜
- **静默等待：** 60分钟
- **最小间隔：** 120分钟 — 不超过每2小时一次

## 如何添加家庭成员档案

为每个家庭成员创建文件 `data/users/{userId}.json`（如何获取userId——见[第11章](11-users.md)）：

妈妈：
```json
{
  "appeals": ["小美", "美美", "美丽"],
  "notes": "妈妈。喜欢烹饪，尤其是烘焙。对坚果过敏。热衷园艺。每个星期天都烤蛋糕。"
}
```

爸爸：
```json
{
  "appeals": ["大强", "强哥", "建国"],
  "notes": "爸爸。工程师。喜欢钓鱼和足球。周六去郊外。什么都修。"
}
```

女儿：
```json
{
  "appeals": ["小雨", "小甜甜", "雨雨"],
  "notes": "女儿，8岁。上二年级。喜欢画画和猫。怕黑。周三上舞蹈课。"
}
```

库佳会记住如何称呼每个人，并在交流中考虑他们的兴趣和习惯。
