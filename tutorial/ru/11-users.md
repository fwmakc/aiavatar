# 11. Профили людей

Профили позволяют боту запомнить, кто есть кто — как обращаться, что знать, какие отношения.

## Как узнать user ID

### Способ 1: через бота

Добавьте в чат бота [@userinfobot](https://t.me/userinfobot). Перешлите ему сообщение человека — он покажет user ID.

### Способ 2: через консоль

Когда кто-то пишет боту, в консоли появляется:
```
[DM] 123456789: Привет!
```
Число после `[DM]` — это user ID.

### Способ 3: через Telegram API

Отправьте любое сообщение боту, затем откройте в браузере:
```
https://api.telegram.org/botВАШ_ТОКЕН/getUpdates
```
Найдите `"from":{"id":123456789}` — это user ID.

## Как узнать chat ID группы

Отправьте сообщение в группу, где добавлен бот, затем:
```
https://api.telegram.org/botВАШ_ТОКЕН/getUpdates
```
Найдите `"chat":{"id":-1001234567890}` — это chat ID (с минусом).

Для имени файла конфига **минус убирается**: `1001234567890`.

## Создаём профиль

Создайте файл `data/users/{userId}.json`:

```json
{
  "appeals": ["Саша", "Санёк"],
  "notes": "Фронтенд-разработчик. Любит React и TypeScript. Пьёт много кофе. По пятницам играет в настолки."
}
```

### Поля

| Поле | Описание | Пример |
|---|---|---|
| `appeals` | Как обращаться к человеку | `["Саша", "Санёк", "Александр"]` |
| `notes` | Что знать о человеке | `"Фронтендер, любит React"` |

Бот будет использовать `appeals` для обращения и `notes` для контекста в разговорах.

## Примеры профилей

### Коллега (для сценария «Макс»)

```json
{
  "appeals": ["Ваня", "Ванька"],
  "notes": "Бэкенд-разработчик на Python. Любит спорить о типах. Фанат Linux. Каждое утро начинает с чтения Hacker News."
}
```

### Член семьи (для сценария «Кузя»)

```json
{
  "appeals": ["Дашенька", "Даша"],
  "notes": "Дочка 8 лет. Учится во 2 классе. Любит рисовать и кошек. По средам — танцы. Боится темноты."
}
```

## Персональная личность в ЛС

Подробно описано в [главе 8](08-personal-chats.md) — там примеры для ребёнка, бабушки и коллеги.

Если кратко: создайте `data/personal_chats/{userId}.json`, чтобы бот общался с конкретным человеком в ЛС по-особенному. Это **не влияет** на общение в группах.

## Отношения

Бот автоматически строит отношения с каждым человеком:

- **Score от −5 до +5** — зависит от того, как человек общается с ботом
- **TIT FOR TAT:** грубишь боту — бот отвечает холодом, помогаешь — теплом
- **5 стадий:** hostile → cold → neutral → warm → intimate

Отношения хранятся в базе данных (`data/bot.db`) и не требуют ручной настройки.
