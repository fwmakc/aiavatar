## Что здесь настроено

### Личность

- **Имя:** Макс — IT-ветеран
- **Язык:** русский
- **Стиль:** дружелюбный коллега с юмором, не формальный ассистент
- **personaStages:** 5 уровней отношений — от врага до друга. Чем лучше отношения, тем теплее и неформальнее общение

### Контент

| Источник | Что даёт | Настройка |
|---|---|---|
| Bash.im | IT-юмор, цитаты | Постится как есть |
| JokeAPI | Программистские шутки | Постится как есть |
| Habr | IT-новости | AI пересказывает в стиле Макса |
| OpenNet | Новости из мира Linux/FOSS | AI пересказывает |
| Securelist | Кибербезопасность | AI пересказывает |
| TechCrunch | Зарубежные IT-новости | Перевод + AI пересказ |
| Ars Technica | Глубокие IT-статьи | Перевод + AI пересказ |
| Hacker News | Обсуждения из мира технологий | Перевод + AI пересказ |

**Викторины** — по 9 темам: от JS до кибербезопасности. Макс задаёт вопросы и оценивает ответы.

**Wellness-напоминания** — глаза, спина, запястья, стресс. Актуально для программистов.

### Расписание

- **Активные часы:** Пн–Пт, 09:00–18:00 — в рабочее время
- **Пауза после тишины:** 45 минут — если в чате тихо 45 минут, можно постить
- **Минимальный интервал:** 90 минут — не чаще раза в полтора часа

## Как настроить под себя

### Сменить специализацию

Замените `specialization` и `interests`:

```json
"specialization": "QA-инженер с 10-летним опытом, знаешь все виды тестирования",
"interests": "тестирование, автоматизация, Selenium, CI/CD, баг-репорты, мемная культура тестировщиков"
```

### Добавить свой RSS

Добавьте объект в массив `feeds`:

```json
{
  "url": "https://your-company-blog.com/rss/",
  "type": "rss",
  "comment": true,
  "weight": 8
}
```

### Добавить прогноз погоды

Scheduled-фиды постятся в указанное время, минуя обычную очередь контента:

```json
{
  "url": "https://api.open-meteo.com/v1/forecast?latitude=55.75&longitude=37.62&daily=temperature_2m_max,temperature_2m_min,weathercode&timezone=Europe/Moscow&forecast_days=1",
  "type": "json",
  "path": "daily",
  "comment": true,
  "commentPrompt": "You are sharing today's weather forecast. Be brief, add a practical tip (umbrella, dress warm, etc). Data: {text}",
  "scheduled": ["07:30"],
  "weight": 3
}
```

- **`scheduled`** — массив времени в формате `"HH:MM"` (локальное время сервера). Фид постится в эти часы, даже если в чате идёт активная беседа
- **`commentPrompt`** — кастомный промпт для AI. `{text}` заменяется на данные из API. Полезно, когда API возвращает структурированный JSON, а не готовый текст
- **`path`** — путь к данным в JSON-ответе. Если данные — массив или объект, AI получит их целиком и сам сформулирует текст

Если несколько scheduled-фидов назначены на одно время — постятся все.

### Изменить расписание

```json
"schedule": {
  "activeHours": { "start": "08:00", "end": "20:00" },
  "activeDays": [1, 2, 3, 4, 5],
  "idleThresholdMin": 30,
  "minIntervalMin": 60
}
```
