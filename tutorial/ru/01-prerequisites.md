# 1. Что понадобится

Перед началом убедитесь, что у вас есть:

## Node.js 18+

Бот написан на TypeScript и запускается через Node.js. Нужна версия 18 или выше.

Проверьте:
```bash
node --version
```

Должно быть `v18.x.x` или выше. Если не установлен — скачайте с [nodejs.org](https://nodejs.org/).

## Telegram-аккаунт

Нужен обычный Telegram-аккаунт. Через него вы создадите бота через @BotFather.

Если Telegram заблокирован в вашем регионе — понадобится прокси или VPN. О настройке прокси для бота — в [главе 11](11-tips.md).

## API-ключ от AI-сервиса

Бот использует нейросеть для общения, анализа тона и генерации контента. Подходит любой сервис, совместимый с Anthropic API или OpenAI API:

| Сервис | URL | Формат | Цена |
|---|---|---|---|
| **Kimi** (Moonshot AI) | `https://api.kimi.com/coding/v1` | `anthropic` | Дешёвый, хороший русский |
| **Anthropic Claude** | `https://api.anthropic.com/v1` | `anthropic` | Дороже, качественнее |
| **OpenAI GPT-4o** | `https://api.openpi.com/v1` | `openai` | Дороже |
| Любой OpenAI-прокси | URL вашего прокси | `openai` | Зависит |

В [главе 3](03-ai-provider.md) мы подробно разберём, как получить API-ключ Kimi — это самый простой и бюджетный вариант.

## Git

Для клонирования репозитория:
```bash
git --version
```

Если не установлен — скачайте с [git-scm.com](https://git-scm.com/).
