# 6. Сценарий: «Кузя» — домовёнок в семейном чате

Настраиваем тёплого, заботливого бота для семейного чата. Кузя — домовёнок, который знает всех членов семьи, даёт советы, напоминает про здоровье и постит полезный контент.

## Как работает этот сценарий

Кузя настраивается как **override для конкретного чата**. Это значит:
- `data/default.json` — общая конфигурация (из главы 5)
- `data/chats/{chatId}.json` — конфигурация Кузи для семейного чата

Так один бот может быть и Максом в рабочем чате, и Кузёй в семейном — с разными личностями.

## Шаг 1. Узнайте chatId семейного чата

Добавьте бота в семейний чат (как это сделать — в [главе 8](08-running.md)).

После добавления отправьте в чат любое сообщение. В консоли бота появится:
```
[Group] chat -1001234567890: ...
```

Число после `chat` — это chatId (с минусом). Для имени файла **минус убирается**, остаются только цифры: `1001234567890`.

## Что здесь настроено

### Личность

- **Имя:** Кузя — домовёнок
- **Стиль:** тёплый, заботливый, с народными поговорками и уменьшительно-ласкательными
- **personaStages:** даже враждебный Кузя всё равно заботится — он же домовёнок, не может иначе

### Контент

| Источник | Что даёт | Настройка |
|---|---|---|
| Bash.im | Лёгкий юмор | Редко (weight 3) |
| JokeAPI (Misc) | Семейные шутки | Чаще (weight 4) |
| Film.ru | Киноновости | AI пересказывает |
| Afisha | Афиша мероприятий | AI пересказывает |

**Викторины** — народные приметы, кулинария, загадки, садоводство, праздники.

**Wellness-напоминания** — глаза, спина, прогулки, здоровый сон, семейная активность.

### Расписание

- **Активные часы:** каждый день, 07:00–22:00 — с утра до позднего вечера
- **Пауза после тишины:** 60 минут
- **Минимальный интервал:** 120 минут — не чаще раза в 2 часа

## Как добавить профили членов семьи

Создайте файл `data/users/{userId}.json` для каждого члена семьи (как узнать userId — в [главе 10](10-users.md)):

Мама:
```json
{
  "appeals": ["Маша", "Машенька", "Мария"],
  "notes": "Мама. Любит готовить, особенно выпечку. Аллергия на орехи. Увлекается садоводством. Каждое воскресенье печёт пироги."
}
```

Папа:
```json
{
  "appeals": ["Саша", "Санёк", "Александр"],
  "notes": "Папа. Работает инженером. Любит рыбалку и футбол. По субботам ездит на дачу. Чинит всё вокруг."
}
```

Дочка:
```json
{
  "appeals": ["Дашенька", "Даша", "Дашута"],
  "notes": "Дочка 8 лет. Учится во 2 классе. Любит рисовать и кошек. Боится темноты. По средам — танцы."
}
```

Кузя запомнит, как обращаться к каждому, и будет учитывать их интересы и привычки в общении.
