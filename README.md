# AIAvatar

Telegram-бот — цифровой двойник с модульной конфигурацией. Общается в группах и личке, постит контент, вмешивается в споры и помнит отношения с людьми.

## Требования

- Node.js 18+

## Установка

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/fwmakc/aiavatar.git
   cd aiavatar
   ```

2. Установи зависимости:
   ```bash
   npm install
   ```

3. Скопируй и заполни `.env`:
   ```bash
   cp .env.example .env
   ```

4. Собери и запусти:
   ```bash
   npm run build
   npm start
   ```

Для разработки:
```bash
npm run dev
```

## Структура конфигов

Все настройки лежат в `data/`, не в коде.

```
data/
  default.json              # Базовая личность бота
  chats/
    -1001234567890.json     # Переопределения для конкретного чата
  users/
    123456789.json          # Социальный профиль человека
  personal_chats/
    123456789.json          # Какой бот для этого человека (только ЛС)
```

### `data/default.json`

Основной конфиг. Поля:
- `name` — имя бота
- `language` — язык общения
- `specialization` — кто он по профессии
- `interests` — культурный код
- `views` — взгляды на мир
- `style` — стиль общения
- `contentSources` — новости, шутки, викторины, челленджи
- `schedule` — тихие часы и дни

### `data/chats/{chatId}.json`

Переопределяет любые поля из `default.json` для конкретного чата. Приоритет: `chats` > `default`.

### `data/users/{userId}.json`

Социальный профиль:
- `appeals` — как к нему обращаться ("Ваня", "Ванька")
- `notes` — что о нём знать

### `data/personal_chats/{userId}.json`

Правила общения в **личной переписке**:
- `persona.specialization` — кто бот для этого человека
- `persona.style` — стиль общения
- `persona.interests` — интересы
- `persona.views` — взгляды

Приоритет: `chats` > `personal_chats` > `default` (но `personal_chats` только для ЛС).

## Функции

- **Контентный движок** — постит новости, шутки, викторины, wellness-напоминания по расписанию
- **Дедупликация** — не постит один и тот же контент дважды
- **Тихие часы** — не спамит ночью и по выходным
- **Социальное вмешательство** — вмешивается, если в чате начинается ссора
- **Отношения** — помнит, кто его друг, а кто нет (`data/relationships.json`, не коммитится)
- **Реакции** — ставит эмодзи на сообщения в группах
