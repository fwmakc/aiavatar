# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please do **not** open public issues for security vulnerabilities.

Instead, report privately via:

- Email: [fwmakcpro@yandex.ru](mailto:fwmakcpro@yandex.ru)
- Or Telegram: [@fwmakc](https://t.me/fwmakc)

We will respond within 48 hours and work on a fix. Once resolved, we will disclose the issue responsibly with credit to the reporter if desired.
